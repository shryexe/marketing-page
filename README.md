## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections Included

1. **Hero Section** - Eye-catching headline with CTA buttons
2. **Features Section** - 6 feature cards with icons
3. **About Section** - Why choose us with benefits
4. **Testimonials Section** - Customer reviews
5. **Pricing Section** - 3 pricing tiers
6. **Footer** - Links and social media

## Customization

- **Colors**: Edit `tailwind.config.js` to change the color palette
- **Content**: Update text in each section component
- **Animations**: Modify Framer Motion props in components
- **Styling**: Use Tailwind utility classes or extend in `index.css`

## Technologies Used

- React 18.2.0
- Vite 5.0.0
- Tailwind CSS 3.3.5
- Framer Motion 10.16.4
- Lucide React (Icons) 0.294.0
